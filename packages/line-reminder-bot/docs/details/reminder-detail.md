# リマインダー詳細表示

## 概要

リマインダーの詳細を表示する機能です。リマインドの内容、実行時刻、削除ボタン、更新ボタンを表示します。

## エントリーポイント

- **トリガー**: リマインダー一覧で送信されたメッセージのボタンを押した際のPostbackイベント
- **判定条件**: Postbackイベントかつ`data`にリマインダー詳細表示コマンドが含まれている場合
- **送信メッセージ形式** : Flexメッセージ(リマインドの内容、実行時刻、削除ボタン、更新ボタン)

## データフロー

1. ユーザーがリマインダー一覧のメッセージのボタンを選択
2. PostbackイベントがLINEから送信される
3. Postbackイベントを検証し、イベントデータからリマインダーIDを抽出
4. ユーザー認証を実行
5. `showReminderDetail()` を呼び出し、リマインダー詳細を取得
6. D1データベースからリマインダーIDに対応するリマインダーを取得
7. 詳細メッセージをLINEに返信
