# リマインダー一覧

## 概要

ユーザーがそのユーザーの全リマインダーを実行時刻順（近い順）に表示します。

## エントリーポイント

- **トリガー**: LINEのリッチメニューからのPostbackイベント（一覧表示コマンド）
- **判定条件**: Postbackイベントかつ`data`に一覧表示コマンドが含まれている場合
- **送信メッセージ形式** : Flexメッセージ(ボタンが5つ縦並び、ラベルはリマインドの内容)

## データフロー

1. ユーザーがリッチメニューの「一覧表示」ボタンを選択
2. PostbackイベントがLINEから送信される
3. Postbackイベントを検証し、イベントデータからユーザーIDを抽出
4. ユーザー認証を実行
5. `showReminderList()` を呼び出し、リマインダー一覧を取得
6. D1データベースからリマインダーを実行時刻昇順で取得
7. 一覧メッセージをLINEに返信

## 注意点

### 詳細なメッセージは表示しない

詳細なメッセージは送信したメッセージのボタンを押した際のPostbackイベントで表示するため、ここではリマインドのテキストのみを表示します。
