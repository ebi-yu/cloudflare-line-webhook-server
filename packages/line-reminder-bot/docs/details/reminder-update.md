# リマインダー更新

## 概要

ユーザーがリマインダー詳細のFlex Messageから更新ボタンを選択すると、該当するリマインダーグループ（同じ`groupId`を持つ全てのリマインダー）を一括更新します。

更新対象はリマインドの内容のみであり、実行時刻は変更しません。

## エントリーポイント

- **トリガー**: LINE Postbackイベント（リマインダー詳細の更新ボタン、クイック操作の更新）
- **判定条件**: Postbackイベントかつ`data`に更新コマンドが含まれている場合
- - **送信メッセージ形式** : テキストメッセージ(更新完了メッセージ)

## データフロー

1. リマインド詳細のFlex Messageの更新ボタンを選択/クイック操作から更新を選択
2. PostbackイベントがLINEから送信される
3. Postbackイベントを検証し、イベントデータから`groupId`を抽出
4. `updateReminderFromLine()` を呼び出し、リマインダーを更新
5. D1データベースから指定`groupId`のリマインダーを更新
6. 更新完了メッセージをLINEに返信
